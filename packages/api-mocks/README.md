# @podman-desktop/api-mocks

Auto-generated vitest mocks for `@podman-desktop/api`, for use in 3rd party Podman Desktop extension tests.

## Overview

This package provides a vitest global setup hook that auto-generates mocks for all namespaces, functions, and classes exported by `@podman-desktop/api`. The mocks are generated by inspecting the `extension-api.d.ts` type definitions.

## Usage in a 3rd party extension

### 1. Install the package

```bash
pnpm add -D @podman-desktop/api-mocks
```

### 2. Configure vitest

In your `vitest.config.ts` (or `vite.config.ts`), add the global setup and alias:

```ts
import { defineConfig } from 'vitest/config';
import { createRequire } from 'node:module';
import { join } from 'node:path';

const require = createRequire(import.meta.url);
// Resolve the path to the generated mock file
const apiMocksPath = require.resolve('@podman-desktop/api-mocks/@podman-desktop/api');

export default defineConfig({
  test: {
    globals: true,
    globalSetup: ['@podman-desktop/api-mocks/setup'],
    alias: {
      '@podman-desktop/api': apiMocksPath,
    },
  },
});
```

### 3. Write tests

All `@podman-desktop/api` functions are automatically mocked with `vi.fn()`:

```ts
import { commands, window } from '@podman-desktop/api';
import { beforeEach, expect, test, vi } from 'vitest';
import { activate } from './extension.js';

beforeEach(() => {
  vi.resetAllMocks();
});

test('activate registers a command', async () => {
  await activate(context);
  expect(commands.registerCommand).toHaveBeenCalledWith('my-extension.hello', expect.any(Function));
});
```

## How it works

The `@podman-desktop/api-mocks/setup` global setup hook:

1. Reads the `extension-api.d.ts` type definitions from `@podman-desktop/api`
2. Extracts all exported namespaces, functions, and classes
3. Generates a `@podman-desktop/api.ts` mock file using a Mustache template
4. Skips regeneration if the generated file is already up-to-date

The generated mock file is committed to the package and shipped as part of the npm distribution.

## Published package

This package is published to npm as `@podman-desktop/api-mocks`.

The `@podman-desktop/api-mocks/setup` export is the vitest global setup function.  
The `@podman-desktop/api-mocks/@podman-desktop/api` export is the generated mock file.
